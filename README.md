# ToySSG

A micro SSG(static site generator) built from scratch with [rhoboro/toypeg](https://github.com/rhoboro/toypeg).

## Getting Started

```bash
# Generate HTML files from Markdown content in `contents/` and output them to the `dist/` directory:
cargo run
```
